## 一、設計總則

### 1.1 品牌定位
一人公司網站作為六大事業體的入口，首頁需傳達「全能、專業、創意」的品牌印象。
每個事業體有獨立視覺語言，但透過統一的導航架構與互動模式維持品牌一致性。

### 1.2 核心設計原則
| 原則 | 說明 |
|------|------|
| **清晰勝於花俏** | 每個頁面只有一個主要行動呼籲（CTA） |
| **數據即介面** | 金融、土木等專業模組以數據視覺化為核心 |
| **遊戲化體驗** | 非專業模組加入微交互與趣味反饋 |
| **一致性底層** | 所有業務線共用同一套 Grid、Spacing、Typography 基線 |

### 1.3 佈局底層（共用）
- **最大內容寬度**: 1280px（桌面），768px（平板），100%（手機）
- **柵格系統**: 12 欄 CSS Grid，gutter 24px
- **圓角體系**: 4px（微型）/ 8px（卡片）/ 12px（模態框）/ 16px（大區塊）
- **陰影層級**: 
  - `sm`: 0 1px 2px rgba(0,0,0,0.05)
  - `md`: 0 4px 12px rgba(0,0,0,0.08)
  - `lg`: 0 8px 30px rgba(0,0,0,0.12)
  - `xl`: 0 20px 60px rgba(0,0,0,0.15)

## 六、共用元件庫

### 6.1 導航系統

**主導航（全域）**
- 位置：固定頂部（height: 64px）
- 左側：Logo「OPC」+ 下拉選單顯示六大事業體
- 右側：深色/淺色切換 + 會員頭像
- 響應式：桌面為完整選單，手機為漢堡選單
- 背景：毛玻璃效果 (`backdrop-filter: blur(12px)`)

**次導航（業務線內）**
- 位置：主導航下方（height: 48px）
- 樣式：Tab 切換，底部滑動底線指示器
- 滾動行為：滾動後隱藏（`sticky` + 向上滾動顯示）

### 6.2 按鈕系統

| 類型 | 樣式 | 用途 |
|------|------|------|
| Primary | 填滿主色 + hover 加深 | 主要行動 |
| Secondary | 邊框 + hover 填滿 | 次要行動 |
| Ghost | 透明 + hover 背景 | 工具列 |
| Danger | 紅色填滿 | 刪除/危險操作 |
| Icon | 圓形 36px，僅圖示 | 工具列/操作列 |
| Large | Primary 的放大版 (48px h) | CTA 按鈕 |

### 6.3 卡片系統

- **預設卡片**: 圓角 8px，背景 (白/深色面)，內邊距 24px，陰影 md
- **點擊卡片**: hover 時 `translateY(-2px)` + 陰影升級
- **數據卡片**: 左上角圖示，中央大數字，底部小字說明
- **商品卡片**: 圖片 + 內容區，無陰影但有邊框

### 6.4 表單元件

- 輸入框：圓角 8px，邊框 1px，focus 時主色邊框 + 微陰影
- 下拉選單：自定義樣式，圓角 8px，選項 hover 高亮
- 開關（Toggle）：iOS 風格，平滑過渡
- 滑桿（Range）：自定義 thumb（圓形 + 陰影）

### 6.5 載入狀態

- **Skeleton 骨架屏**: 所有卡片/列表載入時顯示灰色脈衝動畫
- **Spinner**: 圓環旋轉（與該業務線主色一致）
- **進度條**: 填滿漸層 + 不確定動畫

## 七、深色/淺色主題策略

### 7.1 預設行為
- 首頁：跟隨系統設定（`prefers-color-scheme`）
- 每個業務線可獨立設定預設主題：
  - 量化交易：**強制深色**（數據在暗色上更清晰）
  - AI 工具：**強制深色**（科技感）
  - 經典遊戲：**強制深色**（復古遊戲氛圍）
  - 土木結構：淺色（專業文件感）
  - 二手交易：淺色（電商平台慣例）
  - 線上 PPT：跟隨編輯器設定

### 7.2 切換方式
- 首頁右上角：全域深色/淺色切換
- 每個業務線可獨立切換（覆蓋全域設定）
- 切換動畫：0.3s ease 過渡所有顏色屬性

## 九、動畫與微交互規範

### 9.1 持續時間
| 情境 | 時間 | 緩動函數 |
|------|------|---------|
| hover 效果 | 150ms | ease-out |
| 頁面切換 | 300ms | ease-in-out |
| 模態框進出 | 250ms | ease-out |
| 拖放操作 | 200ms | ease |
| 通知出現 | 400ms | ease-out |
| 主題切換 | 300ms | ease |

### 9.2 常用動畫
- **Fade In**: `opacity 0 → 1`
- **Slide Up**: `translateY(20px) → 0` + fade
- **Scale In**: `scale(0.95) → 1` + fade（模態框）
- **Stagger**: 列表項目依序延遲 50ms 進場
- **Skeleton**: `@keyframes pulse` 背景色來回漸變
